## taichi实现拉普拉斯方程基本解模拟
实现对于理想不可压流体的平面定常无旋流动的一些基本形式的模拟，如点源、点汇、点涡和偶极子。

### 运行
```shell script
python demo.py
```
* 按下`s`/`v`/`d`键后在gui中按下鼠标左键或右键，将在鼠标位置创建一个点源/点涡/偶极子
或强度为负数的点源（点汇）/点涡/偶极子
* 在已经创建的点源/点涡/偶极子附近同时按下鼠标左键和右键将删除该对象
* 直接在点源/点涡/偶极子附近按下鼠标左键将增加其强度的绝对值；若按下右键将减少其强度的绝对值
